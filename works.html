Programming and networks

Graph and List Management Tool

This Windows Forms app in C# features an intuitive interface for administering graphs and lists. Users can create, modify, and delete graph nodes and edges. The application allows choosing between managing integers or strings and applying search algorithms like BFS and DFS to explore the nodes. It also supports list operations, such as inserting elements at various positions and deleting them by value or position.

Football Team
CRUD App

This Windows Forms app in C# provides a comprehensive system for managing a football team. Through a user-friendly interface, users can create, edit, and delete teams, as well as manage individual player details. Additionally, the application offers the capability to view detailed team statistics, automatically calculates the team's total value, and provides the average age of players.

Enterprise Network Simulation

This project done with Cisco Packet Tracer reproduces an enterprise network infrastructure. The company is distributed across four separate locations, interconnected through a unified Internet link using the RIP protocol. Each location comprises two offices, each equipped with a switch to manage connections for multiple PCs and printers. The network design includes dynamic routing facilitated by a DHCP server.
